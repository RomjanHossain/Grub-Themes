### Installation/Update

	- git clone https://github.com/bishwassagar/grub2-deadpool-theme.git
	- cd grub2-deadpool-theme
	- chmod 755 install.sh
	- sudo ./install.sh
	- cd ..
	- rm -r grub2-deadpool-theme

### Done!!!
